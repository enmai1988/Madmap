## Usage

To launch the app:
    1. Simply Install the dependencies
    2. Prep the app for developement
    3. And visit localhost:3000

## Requirements

- Node 6.4.x
- Postgresql 9.1.x

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Running Tests

From within the root directory:

```sh
npm test
```

### Preparing ENV for Developement
```
npm run react-dev
npm run server-dev
npm run eslint-pom
```
